User Data Deletion Instructions for song

Last updated: 2026-06-29

How to Request Deletion

To request deletion of data stored by song, send a deletion request to the app owner using the contact email configured in the Meta Developer app dashboard for song. Include the phrase "song Threads data deletion request" and the Threads username or profile URL related to the request.

What Will Be Deleted

The app owner will delete locally stored Threads API data associated with the requested account from the MoneyMachine reporting workflow, including stored post text, permalinks, timestamps, public media metadata, and collected conversation reply records when present.

Timing

Deletion requests are reviewed and completed within 30 days unless additional information is required to identify the relevant data.

Limitations

song can delete only data stored in its own local workflow. It cannot delete content from Threads itself, Meta systems, search engines, screenshots, browser caches, or third-party archives. To delete content from Threads, manage the content directly in your Threads account.
